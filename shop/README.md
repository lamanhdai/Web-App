## How to use

**Run on develop environment**
- Step 1: run mongo server `mongod`
- Step 2: run mongo database `mongo`
- Step 3: run node server `npm start`
- Step 4: run webpack `npm run dev`
- Step 5: open in browser `localhost:8080`
